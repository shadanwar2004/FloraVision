# FloraVision

A modern and responsive plant showcase website built using **React.js**, **Vite**, **Tailwind CSS**, **Framer Motion**, and **Lucide React**.

FloraVision features a premium dark-glass UI design with animated sections, trendy plant collections, customer reviews, and responsive layouts inspired by modern landing pages.
